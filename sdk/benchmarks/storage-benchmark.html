About This Benchmark

This benchmark compares OPFS (Origin Private File System) vs IndexedDB performance for typical SyncKit storage operations.

Tests include:

  • Write performance (small, medium, large documents)
  • Read performance
  • List operations
  • Delete operations
Note: This benchmark will clear any existing data in the benchmark-idb and benchmark-opfs storage instances. Your actual SyncKit data will not be affected.
